# Motor/Driver Test Build

## Overview

This circuit is assembled to test the functionality of the motor and driver before permanent soldering is completed. It uses the Arduino Uno R3, one TB6616FNG Motor Driver, one DC motor, and one 6V battery pack. When powered, the motor should continuously rotate clockwise.

Digital inputs to the driver are hardwired (i.e., PWM inputs and HIGH digital inputs are tied directly to 3.3V logic voltage, and LOW digital inputs are tied directly to logic GND).

No code is used with the Arduino Uno.
